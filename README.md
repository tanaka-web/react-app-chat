# react-app-chat

app: https://react-app-chat-439c2.web.app/
github: https://github.com/tanaka-web/react-app-chat


## setting

```
git clone https://github.com/tanaka-web/react-app-chat.git
cd react-app-chat
yarn install
```


## deploy local

```
yarn start
```
http://localhost:3000


## deploy production
```
yarn deploy
```
